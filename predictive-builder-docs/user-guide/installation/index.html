Downloading the appÔÉÅ

Choose one of the following ways to download the application, taking into account if you have git or not to use git on your server.

Download all source code via git in the same folder locationÔÉÅ

git clone git@github.com:PredictiveLAB/predictive-builder.git
git clone git@github.com:PredictiveLAB/predictivelab.git

Download the following .zip files, and extract them in the same folder locationÔÉÅ

https://github.com/PredictiveLAB/predictive-builder/archive/refs/heads/main.zip https://github.com/PredictiveLAB/predictivelab/archive/refs/heads/main.zip

Initialize the Python environmentÔÉÅ

From the folder you saved the files, run following command lines

on WindowsÔÉÅ

cd predictive-builder
python -m venv .venv
.venv\Scripts\activate

on LinuxÔÉÅ

cd predictive-builder
python -m venv .venv
source .venv/bin/activate

InstallÔÉÅ

pip install -r requirements.txt
cd ..
cd predictivelab
pip install -e .
cd ..
cd predictive-builder

Run the AppÔÉÅ

From the predictive-builder folder run the following command lines

cd imap01
python manage.py migrate
python manage.py runserver

At this point, you can acces the app from the browser using the url http://localhost:8000

Insert initial data in databaseÔÉÅ

Quit the server with CTRL-BREAK, and execute the following commands in console (from the same location)

python manage.py create_groups
python manage.py create_archs_targets
python manage.py create_factories_eq_sensors
python manage.py create_hyperparams_options

Create super userÔÉÅ

python manage.py createsuperuser

run the server again

python manage.py runserver

and assign the user roles to the superuser in the url http://localhost:8000/admin/auth/user/1/change/
