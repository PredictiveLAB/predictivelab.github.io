class moncon.preprocessing.parameter_extraction.combined_extraction.CombinedExtraction(preprocess_params, chckpt_filepath=None)[source]

Bases: BasePreprocess

Preprocessing method that combines TemporaryDomain and BandFourier by using different columns for each, and using the same parameters “segment” and “overlap” for both. Input data is divided into two by using “temporary_sensors_mask” parameter, which defines which columns are to be processed by TemporaryDomain and which are to be processed by BandFourier; they are preprocessed separately, and then merged.
