class moncon.preprocessing.filters.DeleteTransient(preprocess_params, chckpt_filepath=None)[source]

Bases: BasePreprocess

It finds every time the equipment is shut down and deletes that information. If the value of a sensor in a specific column called ‘filter_col’ is lower than a value ‘min_value’, delete it. It also deletes ‘n_values’ after and before the range of values deleted. :param min_value: Minimum value of ‘filter_col’ to consider the machine as turned on. :type min_value: float :param n_values: Number of values to be eliminated before and after the shutdown, considered transient. :type n_values: int :param filter_col: Column to be used to find the shut downs. :type filter_col: str :param drop_filter_col: If True, the column ‘filter_col’ is deleted after the preprocessing. :type drop_filter_col: bool

class moncon.preprocessing.filters.OutlierRemoval(preprocess_params, chckpt_filepath=None)[source]

Bases: BasePreprocess

Preprocessing for deleting outliers in a dataset with one class (like healthy machine behavior). Outliers are detected using the IQR method for each column, using Q25 - 3*IQR as the lower limit, and Q75 + 3*IQR as the upper limit; then outliers and are deleted.

Outlier removal is only supposed to be used for cleaning a training dataset of one class, if that wheren’t the case it could delete data from another class; which also means, that it is not supposed to be used on testing data for outlier detection or classification. Therefore, it only removes outliers using fit_transform method, as it doesn’t make sense to fit it and use it for transforming a different dataset (like a testing one).

class moncon.preprocessing.methods.StopFilterMov(preprocess_params, chckpt_filepath=None)[source]

Bases: BasePreprocess

Finds every time the machine is shut down, based on ‘filter_col’ values, separates those sections together with ‘n_values’ before and after, considered transient operation; then calculates the moving average on each section where the machine is turned on, and substracts it to that section. Finally, it returns all turned on sections together as a Dataframe. :param datos: Dataframe with the data to be filtered. :type datos: pd.DataFrame :param n_average: Number of values to calculate the moving average with. :type n_average: int :param min_value: Minimum value of ‘filter_col’ to consider the machine as turned on. :type min_value: float :param n_transient: Number of values to be eliminated before and after the shutdown, considered transient. :type n_transient: int :param filter_col: Column to be used to find the shut downs. :type filter_col: str :param drop_filter_col: If True, the column ‘filter_col’ is deleted after the filter. :type drop_filter_col: bool
