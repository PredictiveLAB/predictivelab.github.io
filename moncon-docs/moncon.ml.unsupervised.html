class moncon.ml.unsupervised.autoencoder.AutoCNN(arch_params, chckpt_filepath=None)[source]

Bases: AutoEncoderModel

Autoencoder model based on a CNN architecture. Inherits from base_arch.AutoEncoderModel.

AutoCNN is a model that uses a convolutional neural network (CNN) architecture to encode and decode data, using the reconstruction error to output a KPI between 0 and 1, indicating how anomalous the data is. Transforms the input data into a batch of matrices (windows), where each matrix is time-window with features, and feeds it to Convolutional layers.

Defines the method:

  • init_arch: initializes the model architecture

  • parse_fit_logging:

  • parse_predict_logging:

Requires the following values in arch_params:
  • arch_params[“slope_coef”] (float): Slope (or scale) factor for KPI calculation,

    defines how quickly the KPI grows

  • arch_params[“contamination”] (float): Proportion of outliers in the data set.

  • arch_params[“time_size”] (int): Number of time steps in the input to use in each window.

  • arch_params[“overlap”] (int): Number of time steps to overlap between windows.

  • arch_params[“feature_size”] (int): Number of features in the input.

  • arch_params[“block_num”] (int): number of hidden CNN blocks in encoder and decoder

  • arch_params[“filter_num”] (int): number of filters in the first CNN block (it is doubled in each block)

  • arch_params[“kernel_v”] (int): vertical size of the convolution kernel

  • arch_params[“kernel_h”] (int): horizontal size of the convolution kernel

Requires the following values in train_params (for fit method):

  • train_params[“epochs”] (int): Number of epochs to train the model

  • train_params[“batch_size”] (int): Batch size for training the model
